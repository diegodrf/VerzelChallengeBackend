
## Tecnologias utilizadas
- Visual Studio 2022
- SQL Server 2022
- .Net 7

## Instruções para rodar local
- Criar um banco de dados com o nome **VerzelVehicles**
- Alterar as configurações do projeto localizada em **VerzelChallengeBackend\WebApi\appsettings.json**
	- ConnectionStrings.Default: Informar a connection string do banco de dados
- Executar o script **VerzelChallengeBackend\DatabaseScript.sql** para criação das tabelas e popular o banco de dados para iniciar o projeto.
